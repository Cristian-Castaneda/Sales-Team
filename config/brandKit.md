# OpenClaw Skill — Design Tokens
## Expense 360 / Comisol Brand Guidelines

Use these tokens to generate on-brand images and videos for any platform.
Logo files will be provided separately by the team.

---

## FONTS

| Role | Family | Weight | Notes |
|------|--------|--------|-------|
| Headings | **Markazi Text** | Bold / Extrabold | Serif, Google Fonts |
| Body / UI | **Inter** | Regular (400), Medium (500), Semibold (600) | Sans-serif, Google Fonts |
| Fallback | system-ui, sans-serif | — | — |

## LIGHT THEME

### Backgrounds
| Name | Hex | Usage |
|------|-----|-------|
| Page BG | `#ecf0f1` | Default page background |
| Section BG | `#fcf3f8` | Feature / alternate sections |
| Card BG | `#f5f5f5` | Cards, panels |
| Card BG Blue | `#cfe4fc` | Help/info cards |
| White | `#ffffff` | Content areas, navbars (scrolled) |
| Footer BG | `#031a6b` | Footer (same on both themes) |

### Gradients
| Name | Value | Usage |

## LOGOS

Place logo files in the `/openclaw/logos/` folder:

| File | Use on |
|------|--------|
| `logo_light.png` | Dark backgrounds / hero / dark theme |
| `logo_dark.png` | Light backgrounds / white navbars / light theme |

Logo sizing reference: max-height `40px` desktop, `42px` mobile.
